Enhancements to Fix the Black Space and Improve the Dashboard
1. Fix the Black Space at the Bottom
Why:
The black space occurs because the content doesn’t fill the viewport height (min-h-screen is applied, but the content is too short). The Recent Activity section is minimal, especially for new users with no activity.
Enhancement:
Ensure the main content container stretches to fill the viewport height using min-h-[calc(100vh-64px)] (accounting for the header height).

Add more content to the Recent Activity section to naturally fill the space (e.g., suggested actions, recent job matches, AI insights).
2. Enhance the Recent Activity Section
Why:
The "Recent Activity" section currently shows "No recent activity found," which isn’t engaging for new users. We can make it more dynamic by adding suggested actions, recent job matches, and a daily tip, as we did in a previous enhancement.
Enhancement:
Show a list of suggested actions (e.g., "Create your first resume," "Explore job matches").

Display recent job matches with a "Practice Interview" button linking to the Job Interview Avatar simulator.

Add a "Daily Tip" widget for job-hunting advice.
3. Add More Engaging Content
Why:
To make the dashboard more informative and interactive, we can add sections like a "Job Search Progress" widget, AI-driven insights (e.g., resume score, top skills in demand), and a "Quick Actions" panel to guide users.
Enhancement:
Add a "Job Search Progress" widget showing application milestones.

Include a "Quick Actions" panel with buttons like "Create Resume," "Find Jobs," and "Improve Resume Score."

Add AI-driven insights (e.g., "Resume Score," "Top Skills in Demand").
4. Align with the Cosmic Navigator Theme
Why:
The dashboard should match the Cosmic Navigator theme (dark gradient background, starfield animation, neon accents, glassmorphism) for visual consistency.
Enhancement:
Apply the jobFinderBackground gradient (linear-gradient(to bottom right, hsl(219, 90%, 10%), hsl(260, 90%, 10%))).

Add the starfield animation.

Use glassmorphism for cards (bg-[rgba(255,255,255,0.1)] backdrop-blur-md border border-[rgba(255,255,255,0.2)]).

Apply neon accents (hsl(221.2 83.2% 53.3%)) for buttons and highlights.
5. Add GSAP Animations for Interactivity
Why:
Animations make the dashboard feel dynamic and engaging. A subtle entrance animation for the cards and a fade-in for the Recent Activity section will enhance the user experience.
Enhancement:
Use GSAP to animate the entrance of the overview cards (fade-in and slide-up).

Add a fade-in animation for the Recent Activity section.

// Job Interview Avatar Component
function JobInterviewAvatar({ job }) {
  const [isSpeaking, setIsSpeaking] = useState(false);
  const [userResponse, setUserResponse] = useState("");
  const [feedback, setFeedback] = useState("");
  const questions = [
    `Tell me about a time you used a key skill in a project.`,
    `How do you approach challenges in ${job.title} roles?`,
    `Why are you interested in this position at ${job.company}?`,
  ];
  const [currentQuestion, setCurrentQuestion] = useState(0);

  const handleSpeak = () => {
    setIsSpeaking(true);
    setTimeout(() => setIsSpeaking(false), 2000);
  };

  const handleSubmitResponse = () => {
    setFeedback("Good answer, but try to be more concise. Aim for 1-2 minutes.");
    if (currentQuestion < questions.length - 1) {
      setCurrentQuestion(currentQuestion + 1);
      setUserResponse("");
    }
  };

  return (
    <div className="space-y-4">
      <div className="flex items-center gap-4">
        <div className={`w-16 h-16 rounded-full bg-gray-500 ${isSpeaking ? "animate-pulse" : ""}`}></div>
        <div>
          <p className="text-lg font-semibold">{questions[currentQuestion]}</p>
          <Button onClick={handleSpeak} className="mt-2 bg-[hsl(221.2,83.2%,53.3%)] hover:bg-[hsl(221.2,83.2%,63.3%)] text-black">
            Speak Question
          </Button>
        </div>
      </div>
      <textarea
        className="w-full p-2 rounded-lg bg-gray-800 text-white"
        value={userResponse}
        onChange={(e) => setUserResponse(e.target.value)}
        placeholder="Type your response here..."
      />
      <Button onClick={handleSubmitResponse} className="bg-[hsl(221.2,83.2%,53.3%)] hover:bg-[hsl(221.2,83.2%,63.3%)] text-black">
        Submit Response
      </Button>
      {feedback && (
        <div className="p-4 bg-[rgba(255,255,255,0.1)] rounded-lg">
          <p className="text-sm text-gray-300">Feedback: {feedback}</p>
        </div>
      )}
    </div>
  );
}
